# ETL com Python e OpenAI

Projeto simples de ETL utilizando Python e integração com a API da OpenAI.

## Fluxo ETL

### Extract
Leitura de dados de usuários.

### Transform
Geração de mensagens personalizadas utilizando IA.

### Load
Exibição e salvamento dos dados processados.

## Tecnologias
- Python
- OpenAI API

## Observação
Foi implementado um sistema de fallback para geração de mensagens caso a API esteja indisponível ou sem créditos.
